# TresEnRaya

## Para Inciar la Aplicación

1. Instalar Depoendecias

   ```bash
   npm install
   ```

2. Iniciar la app

   ```bash
   npx expo start
   ```

Una vez inciada la app , se selecciona el emualrdor :
- a : Android
- i : IOS
- w : Web
  
### Nota:
Expo solo ejecuta la aplicación si hay un emulador encendido anteriormente , sino este da error. Puede ser IOS o Android.
